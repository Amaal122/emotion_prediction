# Emotion Prediction (CNN + Webcam)

This project trains a Convolutional Neural Network (CNN) to classify facial emotions from grayscale face images (48×48) and includes a webcam demo that detects faces and predicts the current emotion in real time.

## Folder structure

- `ai_model/`
  - `train.py` — trains the CNN from the dataset folders and saves the model to `ai_model/emotion_model.h5`
  - `predict.py` — opens your webcam, detects faces, and predicts emotion using `ai_model/emotion_model.h5`
  - `emotion_model.h5` — saved Keras model (can be regenerated by training)
  - `dependencies.txt` — package list (see notes below)
- `dataset/`
  - `train/<class>/...` and `test/<class>/...`
  - classes: `angry`, `disgust`, `fear`, `happy`, `neutral`, `sad`, `surprise`

## Prerequisites

- Windows
- Python **3.11** (recommended for TensorFlow compatibility)
- A webcam (for the live demo)

## Setup (Windows / PowerShell)

Create a virtual environment (recommended):

```powershell
cd C:\Users\bougu\emotion_prediction
py -3.11 -m venv ai_model\.venv311
.\ai_model\.venv311\Scripts\Activate.ps1
```

Install dependencies (typical set used by this repo):

```powershell
python -m pip install --upgrade pip
python -m pip install tensorflow opencv-python numpy matplotlib scikit-learn
```

## Train the model

Make sure your dataset is present at:

- `dataset/train/<class>/...`
- `dataset/test/<class>/...`

Then run:

```powershell
cd C:\Users\bougu\emotion_prediction
.\ai_model\.venv311\Scripts\Activate.ps1
python ai_model\train.py
```

Outputs:
- Model: `ai_model/emotion_model.h5`
- Training curves image: `ai_model/training_results.png`

## Run real-time webcam prediction

```powershell
cd C:\Users\bougu\emotion_prediction
.\ai_model\.venv311\Scripts\Activate.ps1
python ai_model\predict.py
```

Controls:
- Press `ESC` in the OpenCV window to exit.

## Notes

- `dataset/` is intentionally **not tracked** in git (it can be large).
- `ai_model/.venv311/` is also **not tracked** in git.
- `ai_model/dependencies.txt` currently reflects what is installed in the local venv at the time it was generated. If you install additional packages, regenerate it with:

```powershell
.\ai_model\.venv311\Scripts\python.exe -m pip freeze --all > ai_model\dependencies.txt
```
